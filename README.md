# ToothProject
歯科医院の診察アプリ

### 使用技術
- python 3.8.5
- vue@3.3.4
- PostgreSQL(16.1)
- FastAPI

### PostgreSQLの設定
1.PostgreSQLはdockerで動かしているため、下記の記事を参考にDBを設定する。
- https://book.st-hakky.com/hakky/try-postgres-on-docker/

2.使用するデータベースはrootとしてToothProject/backendのパスにあるSQL.txtを参考にテーブル作成及びデータを挿入する。
